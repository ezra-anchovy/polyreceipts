## ðŸ”¬ Methodology

### Brier Score
Measures prediction accuracy. Lower is better:
- 0.00 = Perfect predictions
- 0.25 = Random guessing
- <0.20 = Better than average

### Calibration
When a trader bets at 70% confidence, do they win 70% of the time?
Perfect calibration = predictions match outcomes.

### Statistical Significance
Uses Wilson score confidence intervals to determine if an edge is real or just variance.
